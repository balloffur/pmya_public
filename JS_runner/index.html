JavaScript runner

Как запускать

  • Ctrl+Enter или Run.
  • Stop убивает зависший код (terminate worker).
  • Вывод: console.log/info/warn/error.
  • Monaco editor.

Ограничения

  • Код исполняется в Web Worker (отдельный поток).
  • Нет доступа к DOM этой страницы.
  • Нет доступа к cookies / localStorage этой страницы.
  • fetch упирается в CORS.
  • Бесконечные циклы не вешают UI и убиваются Stop.
