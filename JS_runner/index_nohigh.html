JavaScript runner

Как запускать

  • Ctrl+Enter или Run.
  • Stop пересоздаёт sandbox (убивает зависший код).
  • Вывод: console.log/info/warn/error.
  • Нет подсветки синтаксиса, для этого есть версия с Prism

Ограничения

  • Код исполняется в sandboxed iframe (allow-scripts).
  • Нет доступа к DOM этой страницы.
  • Нет доступа к cookies / localStorage этой страницы.
  • fetch упирается в CORS.
  • Бесконечные циклы грузят CPU до Stop/закрытия вкладки.
